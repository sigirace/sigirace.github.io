## 3. Admin Pannel

**Admin defailt vs custom**

1. default

```python
from django.contrib import admin
from [app].models import [Model]

admin.site.register([Model])
```

2. custom

```python
@admin.register(Model)
class ModelAdmin(admin.ModelAdmin):
  filedsets = (
    (None, {"fields": ("username", "password")}),
  	(),...
 )
  list_display = [
    "model_colum1",
    "model_colum2",
    "model_colum3",
    ...
  ]
  list_filter = [
    "model_colum1",
    "model_colum1",
    ...
  ]
  search_fileds = [
    "model_colum1",
  ]
  exclude = (
    "model_colum1",
    "model_colum2",    
  )
```
